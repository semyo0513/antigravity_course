학생 활동 메모

엑셀이 아니어도 괜찮아요. 학생별로 무엇을 했는지 적어둔 기록이면 충분해요. Step 2에서 엑셀로 정리하는 법을 알려드려요.

1.

세특 작성 규칙, MD 파일로 만들기

안티그래비티는 프로젝트 폴더 안에 있는 파일을 참고해서 답변을 만들어요. 그래서 "이런 규칙으로 써줘"라는 나만의 지침서를 미리 만들어두면, AI가 그 규칙을 그대로 지켜서 세특을 작성합니다. 이 지침서가 바로 MD(마크다운) 파일이에요.

꼭 들어가면 좋은 6가지 규칙

  • ✅학생 활동만 근거로 작성 — 활동지에 없는 내용은 쓰지 않기
  • ✅추측 금지 — "~것으로 보임" 같은 짐작성 표현 자제
  • ✅성취기준 반영 — 교육과정 성취기준과 연결되는 표현 사용
  • ✅과장 금지 — 실제보다 부풀리지 않기
  • ✅종결어미 통일 — 문장을 모두 "~함"으로 마무리
  • ✅글자 수 제한 — 1500Byte 이하로 작성
💡 프롬프트가 구체적일수록 결과 품질이 좋아져요. 우리 학교, 우리 교과만의 규칙이 있다면 자유롭게 추가해보세요.
📄 교과세특작성.md 예시 
# 교과세특 작성 규칙

1. 학생의 실제 수행 활동만 근거로 작성한다.
2. 활동 내용 외의 추측성 표현은 사용하지 않는다.
3. 교육과정 성취기준과 연결되는 표현을 사용한다.
4. 과장되거나 부풀린 표현은 사용하지 않는다.
5. 문장의 종결어미는 "~함"으로 통일한다.
6. 전체 글자 수는 1500Byte를 넘지 않는다.

📌 메모장을 열고 위 내용을 입력한 뒤 파일 > 다른 이름으로 저장에서 파일 이름을 교과세특작성.md로, 파일 형식을 모든 파일로 지정해서 저장하면 끝!

2.

학생 활동, 엑셀로 정리하기

다음은 학생들이 실제로 어떤 활동을 했는지 정리한 엑셀 파일이에요. AntiGravity가 이 파일을 읽고 학생 한 명 한 명의 세특을 작성하게 됩니다. 추천하는 열 구성은 번호 · 이름 · 학생수행활동 세 가지예요.

번호이름학생수행활동
1김○○지역 문제 해결 방안을 발표함
2박○○토론에서 다양한 의견을 제시함
3이○○자료를 분석하여 발표 자료를 제작함
💡 활동 내용은 구체적일수록 좋아요. "발표를 잘함"보다는 "지역 문제 해결 방안을 조사해 발표함"처럼 무엇을, 어떻게 했는지 적어주세요.

파일 이름은 자유롭게 정해도 되지만, 이 가이드에서는 학생활동.xlsx로 부를게요.

3.

규칙과 데이터, 한 폴더에 모으기

이제 방금 만든 MD 파일과 엑셀 파일을 하나의 폴더 안에 함께 넣어주세요. AntiGravity는 이 폴더 전체를 프로젝트로 인식해요.

세특프로젝트/
├── prompt/
│   └── 교과세특작성.md
└── excel/
    └── 학생활동.xlsx

📌 폴더를 꼭 이렇게 나눠야 하는 건 아니에요. 두 파일이 같은 폴더 안에 들어있기만 하면 됩니다.

4.

안티그래비티에 새 프로젝트 만들기

  1. 좌측 메뉴에서 New Project 클릭
  2. [Add Folder] 클릭 → Step 3에서 만든 폴더 선택
  3. Turbo mode 켜기
  4. 생성(Create) 버튼 클릭

💡 Turbo mode란? 파일을 읽고 쓸 때마다 "진행할까요?"라고 매번 승인받지 않고, AI가 자동으로 작업을 이어가도록 허용하는 모드예요. 켜두면 훨�씬 빠르게 진행돼요.

📌 파일 접근 권한을 자동으로 허용하는 기능이니, 세특 작업용 폴더를 따로 만들어 사용하는 걸 추천해요.

5.

AI에게 세특 작성 요청하기

프로젝트가 만들어졌다면, 이제 에이전트 채팅창에 아래 문장을 그대로 입력해보세요.

💬 에이전트에게 보낼 프롬프트 
폴더의 교과세특작성.md를 반드시 준수하세요.
학생활동.xlsx를 읽어 학생별 교과세특을 작성하세요.
학생 활동 외의 내용을 추측하지 마세요.
결과는 시트의 [AI가 작성할 열]에 작성하세요.

📌 [AI가 작성할 열] 부분은 실제 엑셀에서 AI가 답을 적었으면 하는 열 이름으로 바꿔주세요. 예를 들어 세특 열의 제목이 "교과세특"이라면 "결과는 시트의 교과세특 열에 작성하세요"라고 적으면 됩니다.

Enter를 누르면 AntiGravity가 MD 규칙과 학생 활동을 함께 분석해서, 학생 수만큼 순서대로 세특을 작성해줘요. 학생이 많다면 시간이 조금 걸릴 수 있으니 잠시 기다려주세요.

부록

💎 Gemini Pro 이상 구독 중이라면

참고로 무료 계정으로도 충분히 사용할 수 있어요. 다만 Gemini Pro 이상을 구독 중이라면 토큰 한도가 넉넉해서 더 풍부한 자료를 함께 넣어줄 수 있어요. 프로젝트 폴더에 아래 자료를 추가해보세요.

  • 교육과정 PDF
  • 평가계획(수행평가 루브릭)
  • 예시 세특 자료

자료가 많아질수록 AI가 참고할 맥락이 늘어나서, 더 정교한 세특을 써줄 수 있어요.

🚀 나이스(NEIS)에 빠르게 붙여넣기

세특 작성이 끝났다면 나이스에 옮겨 적어야겠죠? 한 명씩 복사해서 붙여넣는 대신, 모두의 복붙(Copy Manager)이라는 도구를 쓰면 훨씬 빨라져요.

  • 🖱️ 엑셀에서 여러 학생의 세특 영역을 통째로 드래그해서 복사(Ctrl+C)하면 자동으로 셀 단위로 인식
  • ⌨️ 한 글자씩 입력하는 매크로가 아니라 클립보드 기반 붙여넣기(Ctrl+V) 방식이라 한글 깨짐이나 오타 걱정이 적음

사용법

  1. 엑셀에서 붙여넣을 세특 내용을 복사(Ctrl+C)
  2. 프로그램에서 탭 이동 횟수를 2회로 설정
  3. 시작 버튼 클릭
  4. 나이스 입력창에 커서를 올려두면 자동 입력 시작
